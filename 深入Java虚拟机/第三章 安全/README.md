## 类装载器体系结构
* Java 沙箱中，类装载器体系机构是第一道防线。是由于类装载器将代码（可能恶意/漏洞）装载入jvm 。类装载器体系结构在三方面对Java 沙箱起作用。

>> 1、防止恶意代码干涉善意代码。（不同的类装载器提供不同的命名空间）

>> 2、守护了被信任的类库边界。

>> 3、将代码装入某类（保护域），该类确定代码可进行的操作。
